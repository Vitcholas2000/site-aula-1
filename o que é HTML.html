O que é Html?

A Tag HTMl é uma etique usada para determina onde começa e onde termina o código.

O que é Head?

Ele compreende as informações que vem dentro do documento, transformado essas informações em título do site.

Exemplo de código

< head >

    < meta charset = "uft-8" >

    < title > ESSE É O MEU TITULO DO SITE.< /title >

< /head >

O que é Body?

É onde fica todo o conteúdo do site, como por exemplo a partir do subtitulo "Tas do Html", já faz parte dentro do Body.

Exemplo de código

< body >

    < p > Bom dia essa parte está dentro do body < /p >

< /body >

O que é o Header?

O Header é o cabeçalho ou o Sumário, ele faz parte do código que faz suporte da introdução do Site, sendo navegacional.

O que é Section?

Representa a seção genérica contida no documento HTML, geralmente com o título.

< section >

    < p > Essa menssagem faz parte da Seção agora < /p >

< /section >

O que é a DIV?

É um elemento de divisão que é um container genérico para conteúdo de fluxo, que de certa forma não representa, mas pode ser utilizado para agrupar elementos para fins de estilos.

< div >

    < p > Pronto minha menssagem está dentro do DIV, essa é a estrutura. < /p >

< /div >

O que é Footer?

O Footer é um rodapé, o rodapé como todo rodapé contem informações do autor do site, direitos autorais e fontes.

< footer >

    < p > Pronto minha menssagem está dentro do Footer, essa é a estrutura. < /p >

< /footer >

O que é H1? E o que acontece ao usar H1 até h7?

H1 é um subtitulo usado em textos, mas ao usarmos H1, H2, H3 constantemente, o código irá parar no H7 Porque? Porque o código não consegue diminuir mais a fonte do que já, então ele dar erro.

O que é HR?

Representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção). Nas versões anteriores do HTML, representava uma linha horizontal. Pode continuar sendo exibida como uma linha horizontal nos navegadores.

< hr / >

    < p > Aparecerá uma linha em cima dessa frase dividindo o site. < /p >

O que é o P?

É o parágrafo.

O que é UL

Ul é a tag para back space, para dar não deixar grudado na margem do programa.

    O que é o IMG src?

    É para adicionar imagem no site.

    O que é o Button?

    É para representar um botão clicavel no site.

    O que é o Strong?

    É para deixar a frase em negrito ou a palavra.

    O que é o Link?

    Especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.

    O que é o label?

    Representa uma legenda para um item em uma interface de usuário.

    O que é o Meta?

    Define qualquer informação de metadados que não podem ser definidos por outros elementos HTML.

    O que é o NAV?

    Representa uma seção de uma página que aponta para outras páginas ou para outras áreas da página, ou seja, uma seção com links de navegação.

    O que é o Progress?

    É um elemento HTML progress ()

    É usado para visualizar o progresso de uma tarefa. Embora as especifidades de como é mostrado ficam a cargo do desenvolvedor, tipicamente, é mostrado como uma barra de progresso.

    O que é o ol?

    Representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos.

    O que é o Output?

    O elemento de saída () é um elemento no qual um site ou aplicativo pode injetar os resultados de um cálculo ou o resultado de uma ação do usuário.
